## Color Palette (from Brand Guidelines)

| Token | Light Mode | Dark Mode | Usage |
|---|---|---|---|
| Accent (Teal-Green) | `#36B489` | `#48C59C` | CTAs, badges, highlights |
| Secondary (Electric Blue) | `#38ACCF` | `#64DAE3` | Secondary accents |
| Background | `#F4F6F9` | `#050D1A` | Page background |
| Background 2 | `#FFFFFF` | `#0A1628` | Card backgrounds |
| Text Primary | `#0A1628` | `#E8EDF5` | Headlines, body |

## Typography (from Brand Guidelines)

| Use | Font | Weight |
|---|---|---|
| Headings | Montserrat | 700–900 (Bold/Black) |
| Subheadings | Montserrat | 400–600 (Regular/SemiBold) |
| Body text | Inter | 300–500 |

All tokens are defined as CSS custom properties in [app/src/app/globals.css](../app/src/app/globals.css) and mapped to Tailwind in [app/tailwind.config.ts](../app/tailwind.config.ts).
